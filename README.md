# Burn Bright

Burn Bright is a fictional fashion brand landing page built with React and Vite.
The site is designed to feel atmospheric and editorial-first, using Tailwind CSS
and the palette `#ec2227`, `#0e120c`, `#6e0e10`, and `#0b1238`.

## Run locally

- Install dependencies: `npm install`
- Start the dev server: `npm run dev`
- Build for production: `npm run build`

## Notes

- The brand name is styled to use Adore You if the font is available in the environment.
- The current version focuses on mood, visual identity, and responsiveness over commerce.
